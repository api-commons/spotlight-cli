# Function to validate if a contract changed between commits

**User story.**
As a developer, I can be able to be warned when a field is removed from an existing OpenAPI request or response object, so that I can avoid API contract changes.

**Is your feature request related to a problem?**
Avoid API contract changing, it sould be nice to choose what kind of changes are allowed and forbidden. Thinhs like field added or field removed.

**Describe the solution you'd like**
A function to validate OpenAPI objects between commits.
