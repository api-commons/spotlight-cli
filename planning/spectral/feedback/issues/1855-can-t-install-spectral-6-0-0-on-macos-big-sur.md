# Can't install Spectral 6.0.0 on macOS Big Sur

**Describe the bug**

When I tried to install Spectral 6.0.0 on macOS Big Sur 11.6, it shows up as a document, rather than a pkg or dmg file. When I try to open this file, my Mac first tries to open it in TextEdit, and then I get an error that "spectral-macos cannot be opened because it is from an unidentified developer. macOS cannot verify that this app is free from malware."

**To Reproduce**

1. Download spectral-macos from https://github.com/stoplightio/spectral/releases/tag/v6.0.0
2. Click on the downloaded file in Chrome or in Finder.
3. See error

**Expected behavior**
Expected behavior was that I could download spectral-macos, open the file, and install Spectral 6.0.0
