# No Docker image tagged for 5.7.x, 5.8.x

Although there are Docker images tagged with 5.4, 5.5, 5.6, there are none for the later releases.

There is an image tagged `5` but it is unclear what version that would be. This is problematic for a CI environment, which relies on stable images that don't change automatically.
