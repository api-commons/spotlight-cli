# Aliases should be inherited across rulesets

**User story.**
As a spectral user, i should be able to use aliases from an extended ruleset within a ruleset i'm authoring.

**Is your feature request related to a problem?**
Spectral will come with a set of aliases for common OpenAPI components, that can then be used by rulesets extending it.
