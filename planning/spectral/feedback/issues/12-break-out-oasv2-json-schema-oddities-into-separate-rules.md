# Break out OASv2 JSON schema oddities into separate rules

We will have some separate validation rules that can be used to enable/disable validations for specific exceptions that OAS makes to their JSON schema support. These exceptions are documented in the  [OASv2](https://github.com/OAI/OpenAPI-Specification/blob/master/versions/2.0.md#schemaObject) specification.
