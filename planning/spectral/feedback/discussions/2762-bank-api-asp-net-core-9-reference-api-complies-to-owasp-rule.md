# Bank API - ASP.NET Core 9 reference API - Complies to OWASP ruleset and 2 others

Bank API is a modern API reference project built with [ASP.NET](http://asp.net/) Core 9.0 Minimal APIs. It is making extensive use of Spectral by leveraging 3 rulesets:

- owasp: https://github.com/stoplightio/spectral-owasp-ruleset?tab=readme-ov-file#spectral-owasp-api-security
- custom: https://github.com/erwinkramer/bank-api/blob/main/.spectral/ruleset.bank.yml
- default oas: https://docs.stoplight.io/docs/spectral/4dec24461f3af-open-api-rules

See the complete source code here: https://github.com/erwinkramer/bank-api?tab=readme-ov-file#bank-api-
