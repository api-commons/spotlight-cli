# Would be nice to include filename into the validation process so you can create rules based on the filename or to compare content of the file with the filename

Few usecases:
1) create rules that would apply to specific file names: e.g. separate rules for latest versions of the schema and non latest (based on the file name)
2) be able to compare info.name with the filename
3) be bale to compare that version in filename corresponds to a version inside schema
and so on
